# Shopping List App

A mobile-first shopping list app with optional per-trip budget tracking.
Designed for fast, one-handed use while shopping.

## Features

- Create multiple shopping trips
- Maintain separate item lists per trip
- Optional budget per trip
- Optional price per item
- Remaining budget indicator
- Over-budget support
- Persistent data via localStorage
- Mobile-focused UI

## Tech Stack

- Vite
- React
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```
